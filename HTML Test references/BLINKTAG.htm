The HTML blink tag

Notice: This tag contains flashing lights that strobe at a speed of 67%. If you are highly prone to seizures, EXIT NOW

If this tag still works, the text will begin blinking. The text has been hidden in a details tag as a protection system for people with epilepsy.

This tag does not work in most browsers

Unsupported browsers:

Firefox 23 and up

Any version of Internet Explorer (1-11)

Note: I have applied JavaScript and CSS to make this tag work. Without it, the tag wouldn't show up in modern browsers.

Why it was removed

This tag wasn't a very important element of HTML, and it was considered unprofessional, distracting, and annoying. It was deprecated, but can still work with CSS and JS
